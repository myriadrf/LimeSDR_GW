CNTRL_EV_STATUS
^^^^^^^^^^^^^^^

`Address: 0xf000d000 + 0x54 = 0xf000d054`

    This register contains the current raw level of the cntrl_isr event trigger.
    Writes to this register have no effect.

    .. wavedrom::
        :caption: CNTRL_EV_STATUS

        {
            "reg": [
                {"name": "cntrl_isr",  "bits": 1},
                {"bits": 31}
            ], "config": {"hspace": 400, "bits": 32, "lanes": 4 }, "options": {"hspace": 400, "bits": 32, "lanes": 4}
        }


+-------+-----------+----------------------------------+
| Field | Name      | Description                      |
+=======+===========+==================================+
| [0]   | CNTRL_ISR | Level of the ``cntrl_isr`` event |
+-------+-----------+----------------------------------+

CNTRL_EV_PENDING
^^^^^^^^^^^^^^^^

`Address: 0xf000d000 + 0x58 = 0xf000d058`

    When a  cntrl_isr event occurs, the corresponding bit will be set in this
    register.  To clear the Event, set the corresponding bit in this register.

    .. wavedrom::
        :caption: CNTRL_EV_PENDING

        {
            "reg": [
                {"name": "cntrl_isr",  "bits": 1},
                {"bits": 31}
            ], "config": {"hspace": 400, "bits": 32, "lanes": 4 }, "options": {"hspace": 400, "bits": 32, "lanes": 4}
        }


+-------+-----------+-----------------------------------------------------------------------------+
| Field | Name      | Description                                                                 |
+=======+===========+=============================================================================+
| [0]   | CNTRL_ISR | `1` if a `cntrl_isr` event occurred. This Event is triggered on **falling** |
|       |           | edge.                                                                       |
+-------+-----------+-----------------------------------------------------------------------------+

CNTRL_EV_ENABLE
^^^^^^^^^^^^^^^

`Address: 0xf000d000 + 0x5c = 0xf000d05c`

    This register enables the corresponding cntrl_isr events.  Write a ``0`` to this
    register to disable individual events.

    .. wavedrom::
        :caption: CNTRL_EV_ENABLE

        {
            "reg": [
                {"name": "cntrl_isr",  "bits": 1},
                {"bits": 31}
            ], "config": {"hspace": 400, "bits": 32, "lanes": 4 }, "options": {"hspace": 400, "bits": 32, "lanes": 4}
        }


+-------+-----------+-------------------------------------------------+
| Field | Name      | Description                                     |
+=======+===========+=================================================+
| [0]   | CNTRL_ISR | Write a ``1`` to enable the ``cntrl_isr`` Event |
+-------+-----------+-------------------------------------------------+
